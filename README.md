* 云盘
    * 使用数据库
        * MySQL
            * 选用使用原因
                * 小
                * 稳定
                * 一般场景都能满足
                * 开源免费
                * 比较熟悉
    * 秒传功能
        * SHA1
            * CRC，MD5，SHA1都差不多，所以选了最常用的SHA1
    * 分块上传
        * Redis
            * 因为只需要保留客户端传来的分块文件，存储量不会太大
            * 需要存取效率较高，可以及时和客户端交流
